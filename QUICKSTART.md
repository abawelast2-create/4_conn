# ⚡ البدء السريع - Quick Start

## 🚀 ابدأ في 3 خطوات فقط!

### الخطوة 1: التثبيت
```bash
pip3 install -r requirements.txt
```

### الخطوة 2: الإعداد
```bash
nano .env
```

**ضع فيه:**
```
EMAIL_USER=admin@yourdomain.com
EMAIL_PASSWORD=YourEmailPassword
```

> 💡 **من Hostinger Business Email:** استخدم البريد وكلمة المرور مباشرة (بدون App Password)

### الخطوة 3: التشغيل
```bash
./start.sh
```

## ✅ الآن:
- افتح: **http://localhost:8765/document-creator.html**
